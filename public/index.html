Building with Hugo

January 30, 2023
This site is built using a static site generator called Hugo. Hugo is a command line tool written using the Go language that takes Markdown or HTML pages and builds a website using a theme you specify. The resulting site consists of static HTML pages and so it loads ridiculously quickly and the pages are much smaller than pages built with a CMS. I am using a theme for the site based on the Poison theme which is itself based on another theme called Hyde.
